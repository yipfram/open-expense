# Open-expense

Milestone 0 bootstrap for V1 using Next.js (App Router), TypeScript, and pnpm.

## Requirements
- Node.js 20+
- pnpm 10+

## Quick Start
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the local dev server:
   ```bash
   pnpm dev
   ```
3. Open `http://localhost:3000`.

## Quality Commands
- Lint: `pnpm lint`
- Format check: `pnpm format:check`
- Format write: `pnpm format`
- Test: `pnpm test`
- Type check: `pnpm typecheck`

## Environment
Copy `.env.example` to `.env` and fill values as needed for later milestones.
